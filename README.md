# Dharma Infra

Premium residential apartment developers in Kukatpally, Hyderabad.

## 🌐 Live Site

**[https://karthikgannamaneni.github.io/DharmaInfra/](https://karthikgannamaneni.github.io/DharmaInfra/)**

## 🛠️ Tech Stack

- TypeScript
- Vite
- CSS3 (Custom animations & Liquid Glass effects)

## 🚀 Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
src/
├── components/     # UI components
├── data/          # Static data (projects, services, etc.)
├── types/         # TypeScript types
├── utils/         # Utilities (animations, smooth scroll, etc.)
├── main.ts        # Entry point
└── style.css      # Global styles
```
